# Linux time Command
-------------------
### Command Introduction (命令介绍)
> ****
### Command Format and Options (命令格式和选项)
```
#time --help
Usage: time [-apvV] [-f format] [-o file] [--append] [--verbose]
       [--portability] [--format=format] [--output=file] [--version]
       [--help] command [arg...]
```
### Command Example (命令范例)
```

  time

  See how long a command takes.

  - Time "ls":
    time ls


```
