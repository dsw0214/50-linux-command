# Linux exit Command
-------------------
### Command Introduction (命令介绍)
> **exit -- The Linux exit command is used to exit the current shell**
### Command Format and Options (命令格式和选项)
```
#exit 
```
### Command Example (命令范例)
```
  # exit                                                                                                                                                                                                          - exit
  logout
  There are stopped jobs. 

```
