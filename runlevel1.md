# Linux runlevel1 Command
-------------------
### Command Introduction (命令介绍)
> ****
### Command Format and Options (命令格式和选项)
```
#runlevel1 --help
runlevel [OPTIONS...]

Prints the previous and current runlevel of the init system.

     --help      Show this help
```
### Command Example (命令范例)
```
	runleve1
	- View the current runlevel
	  runleve1 
```
